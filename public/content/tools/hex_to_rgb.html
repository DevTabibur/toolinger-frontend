The Fundamentals: Understanding Color Models

This section explains how HEX and RGB work. Understanding these models helps you choose colors with accuracy and confidence.

What Is the Difference Between HEX and RGB Color Models?

HEX and RGB describe the same color in two different ways. Designers and developers use both depending on their workflow. RGB shows color with numbers, while HEX uses a shorter text code. Knowing how they connect helps you switch between both formats easily.

Decoding the RGB Color Model (Red, Green, Blue)

RGB builds every color by mixing red, green, and blue light. Each color channel ranges from 0 to 255.

  • A low value makes the color darker.
  • A high value makes the color brighter.

Decoding the HEX Color Model (Hexadecimal Triplet)

HEX expresses RGB values using base-16 numbers. Instead of three decimal numbers, you get a neat code like #1A2B3C.

  • First pair = Red
  • Second pair = Green
  • Third pair = Blue

How to Manually Convert HEX to RGB (The Mathematical Formula)

Sometimes you may want to convert a HEX code without using a tool. The process looks complex at first, but it’s simple once you understand the pattern. Every pair of HEX digits represents a number in base 16, and you convert it into base 10.

To convert, split the HEX code into three pairs: RR, GG, BB.

Then convert each pair using the formula:

(First Digit × 16¹) + (Second Digit × 16⁰)

Let’s convert #A52A2A (a well-known brown color):

  • A5 → (10×16) + 5 = 165
  • 2A → (2×16) + 10 = 42
  • 2A → (2×16) + 10 = 42

So, the RGB result becomes: RGB(165, 42, 42)

Understanding Alpha: How to Use HEX Codes With Transparency

Many designers work with transparency, especially in UI and app design. That’s where 8-digit HEX codes come in. They follow this format:

#RRGGBBAA

The last pair (AA) controls transparency:

  • 00 = fully transparent
  • FF = fully solid

To convert the alpha pair, convert HEX to decimal, then divide by 255 to get a value between 0.0 and 1.0.

Example: #FF000080

  • Red: FF → 255
  • Green: 00 → 0
  • Blue: 00 → 0
  • Alpha: 80 → 128 → 128 ÷ 255 ≈ 0.5

Final result: rgba(255, 0, 0, 0.5)

What Is 3-Digit HEX Shorthand and How Is It Converted?

Some HEX codes use a shorter format like #RGB. This shorthand expands each digit to two digits.

Example:

  • #F00 → #FF0000
  • #ABC → #AABBCC

JavaScript: Convert HEX to RGB

function hexToRgb(hex) {

  hex = hex.replace("#", "");

  if (hex.length === 3) {

    hex = hex.split("").map(c => c + c).join("");

  }

  const r = parseInt(hex.substring(0, 2), 16);

  const g = parseInt(hex.substring(2, 4), 16);

  const b = parseInt(hex.substring(4, 6), 16);

  return { r, g, b };

}

Python: Convert HEX to RGB

def hex_to_rgb(hex_code):

    hex_code = hex_code.replace("#", "")

    if len(hex_code) == 3:

        hex_code = "".join([c*2 for c in hex_code])

    r = int(hex_code[0:2], 16)

    g = int(hex_code[2:4], 16)

    b = int(hex_code[4:6], 16)

    return (r, g, b)

Frequently Asked Questions (FAQ)

Which Is Better to Use: HEX or RGB, and Why?

Both formats work well, but they shine in different tasks. HEX keeps your CSS neat because it uses fewer characters. RGB and RGBA give you more control when you need effects like transparency or dynamic color changes in JavaScript. Use HEX for static design and RGB for flexible styling.

What Are the Minimum and Maximum Values for RGB?

RGB uses three numbers, and each number ranges from 0 to 255.

  • 0 means no light for that color.
  • 255 means full brightness.

What Is the HEX Code for Black and White?

Two of the most common HEX codes are:

  • Black: #000000 → RGB(0, 0, 0)
  • White: #FFFFFF → RGB(255, 255, 255)
